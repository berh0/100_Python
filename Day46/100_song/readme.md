## Bilboard Hot 100

# Objective 
Using BeautifulSoup, scraping the top 100 songs(https://www.billboard.com/charts/hot-100/%7Bdate%7D/) on the date user chose into a Python List.Then creating a spotify playlist using spotipy with that list.
